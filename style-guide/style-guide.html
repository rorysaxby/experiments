The Style Guide

This style guide is designed to contain all elements of the website being developed, it has been structured to contain all of the global common elements at the top with the more specific components further down. It is wrapped within the site's common header, footer and navigation sections. From this page and the other layouts created all pages on the website should be able to be constructed.

NB. When initially created, this style guide is classified as unstyled.
